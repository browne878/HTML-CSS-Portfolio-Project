Support Bot

Our Support Bot is custom made to fit your needs.

Our goal with this bot is to improve response times and allow the relevant memebers of your team to handle the issues assigned to them.

We also wanted to be able to store support queries to be reopened if an issue reoccurs.

Support Bot

Features

  • Easy Management
  • Automated Questions
  • Tickets Logged
  • Embedded Commands
  • Easy to Use
  • Improves Response

Compatibility

This bot is compatible with any server. All it takes is a short amount of time to customise the bot to your needs and you are good to go!

If there are any features you would like to see or that you need for your server, let us know! We are happy to work with anyone.

Customizable

This bot is as customisable as we could make it! We want you to have full control of what features are enabled and disabled.

We even allow you to customise the features! If you want more, let us know and we will get to work!

Rcon Bot

Our Rcon Bot was designed to address issues and make life easier.

We decided to develop this bot to reduce the complexity of handling issues. Our goal was to make sending commands easy and simplify some admin duties.

We also added RCON commands to discord slash commands!

RCON Bot

Features

  • Player Management
  • Automated Commands
  • Command Logging
  • Embedded Commands
  • Easy to Use
  • Command Tiers

Compatibility

This bot is compatible with any game that uses RCON for commands. You can select the game(s) during the setup process.

If there are any custom commands you need let us know! We will arrange a discussion to work out how to apply what you need.

Customizable

The bot has all the regular RCON commands as well as some of our custom commands to make life just a little easier!

We can add additional custom commands and run regular polls and suggestions in our discord!

Event Bot

Our Event bot was designed to make organising community events easier.

Our goal with this bot was to allow admins to prepare events with only 1 or 2 commands.

We also wanted the bot to calculate the results of polls and suggestions so we decided to Automate this process.

Event Bot

Features

  • Automated Events
  • Timed Events
  • Automated Poll Results
  • Embedded Commands
  • Easy to Use
  • Automate Results

Compatibility

This bot is compatible with any server. It will work great with any type of community and will make organising events much easier.

This is a plug and play bot. Once it's invited you can begin to schedule events!

Customizable

The bot itself is not very customizable. However, we hold weekly polls, based on community suggestions, for what features to add.

Any feature of the bot can be enabled or disabled at will via the built in command.
